## 一、自定义详情页（Custom Detail Page）

### 1.1 功能概述

自定义详情页是简道云提供的**组件化、个性化数据详情展示页面**，用于替代系统默认详情页。它允许管理员围绕单个业务对象搭建"专属工作台"，将表单信息、关联记录、统计分析图表、常用操作整合在一个页面中。

**核心价值：**
- 信息集中展示，减少页面跳转
- 支持边查看边操作（如填写跟进记录、更改状态）
- 灵活布局，贴合不同业务场景

**来源：** https://hc.jiandaoyun.com/doc/22707, https://hc.jiandaoyun.com/doc/23299

### 1.2 设置入口与创建方式

1. 进入表单编辑页面 → 「扩展功能 >> 数据详情」→ 点击「升级为自定义详情页」
2. 系统将自动创建 1 张自定义详情页，并进入**页面设计器**
3. 页面设计器提供 **4 种预设模板**，可从模板快速创建
4. 支持自由拖拽组件、灵活设计布局

**移动端支持：**
- 在页面设计器中切换至「移动端」标签页
- 两种创建方式：
  - **从电脑端创建**：同步当前电脑端已有组件及配置（部分组件配置始终与电脑端保持同步，支持手动解除同步）
  - **空白创建**：独立设计移动端页面
- 对于同步组件，删除一端组件仅解除同步关系，不会同步删除另一端

**来源：** https://hc.jiandaoyun.com/doc/22707, https://hc.jiandaoyun.com/doc/23383

### 1.3 可用组件完整列表

自定义详情页提供以下**内容组件**：

| 组件名称 | 功能描述 | 关键特性 |
|---------|---------|---------|
| **简报卡片** | 以卡片形式展示关键数据摘要 | 支持设置简报标题和简报字段；数据源可选"关联的主表数据"或"应用内表单数据"；支持过滤条件和排序；不受字段显隐规则影响 |
| **阶段条** | 可视化展示流程/状态推进进度 | 支持一键更改阶段状态；阶段字段不支持设置了数据联动的下拉框/单选按钮组，或设置了关联其他表单的下拉框 |
| **表单字段** | 直接展示当前表单的字段数据 | 可自定义展示哪些字段；支持样式设置 |
| **数据填报** | 在详情页内直接填写并提交新数据 | 适用于快速填报跟进记录、联系人、订单等；无需跳转页面 |
| **数据表格** | 以表格形式展示关联数据 | 支持查看和管理相关联的客户、任务等数据；支持快捷操作 |
| **列表** | 以列表形式展示数据 | 适用于简洁的数据展示场景 |
| **多标签页** | 将多个组件组织在不同标签页中 | 支持分组展示，提升页面组织性；移动端同步时除外 |
| **图表** | 在详情页中嵌入统计图表 | 支持指标卡、柱状图、堆积柱状图、条形图等；支持维度与指标配置；支持条件格式 |
| **嵌入页面** | 嵌入外部页面或仪表盘 | 支持iframe方式嵌入 |
| **图片** | 展示静态图片 | 用于装饰或说明 |
| **自定义按钮** | 在详情页中放置操作按钮 | 可触发新增数据、编辑数据、执行智能助手等动作 |

**简报卡片详细配置：**
- **数据源类型：**
  - 关联的主表数据：展示当前数据在指定主表中相关联的一条主数据
  - 应用内表单数据：支持进一步设置过滤条件和排序来限定具体使用哪条数据
- **简报标题：** 开启后展示数据标题，支持设置数据标题
- **简报字段：** 支持的字段类型包括：单行文本、数字、日期时间、单选按钮组、下拉框、成员单选、部门单选、定位、地址、流水号、关联数据、手机、计算、提交人、提交时间、更新时间、微信昵称/OpenID、扩展字段、流程状态
- **重要：** 简报标题和简报字段设置不受字段显隐规则影响，内容始终展示

**图表组件详细配置：**
- **指标卡：** 展示单一关键指标数值
- **柱状图：** 
  - 普通柱状图：每个指标独立显示为一组柱子（1个维度 + ≥1个指标）
  - 堆积柱状图：同类别多个指标堆叠在同一根柱子中（1个维度 + ≥2个指标）
  - 维度字段决定横轴分类（支持1-2个维度），指标字段决定柱子高度
- **条形图：** 类似柱状图但方向为水平
- **通用样式设置：** 支持条件格式等

**来源：** https://hc.jiandaoyun.com/doc/23032, https://hc.jiandaoyun.com/doc/25850, https://hc.jiandaoyun.com/doc/22707

### 1.4 数据绑定方式

- **当前表单数据：** 组件默认绑定当前查看的数据记录
- **关联数据：** 通过关联数据字段、关联子表等方式绑定关联表单数据
- **过滤条件：** 支持对应用内表单数据设置过滤条件和排序规则
- **数据权限：** 组件遵循成员对表单的字段权限和数据权限
- **打印行为：** 打印时打印"成员有可见权限的字段"，而非仅打印详情页中展示的表单字段
- **编辑行为：** 点击详情页右上角「编辑」按钮，可在弹出的数据编辑窗口内编辑数据

**来源：** https://hc.jiandaoyun.com/doc/22707, https://hc.jiandaoyun.com/doc/23032

### 1.5 典型应用场景

1. **客户管理：** 查看客户画像 + 历史跟进记录 + 联系人 + 订单数据，直接在详情页填报跟进记录
2. **项目管理：** 查看项目详情 + 阶段条推进状态 + 数据表格管理关联任务
3. **商机管理：** 阶段条一键更改推进状态 + 数据填报快速填报跟进记录
4. **商品管理：** 商品概貌 + 库存数据 + 销售趋势图表

**来源：** https://hc.jiandaoyun.com/doc/23299

## 二、自定义按钮（Custom Button）

### 2.1 功能概述

自定义按钮支持管理员将复杂业务操作设计为一个按钮事件，成员通过点击按钮即可执行对应的表单操作，完成业务处理。自定义按钮提供更便捷的业务操作方式，全面提升工作效率。

**来源：** https://hc.jiandaoyun.com/doc/16221

### 2.2 设置入口

进入要设置按钮的表单 → 「扩展功能 >> 自定义按钮 >> 新建自定义按钮」→ 进入自定义按钮添加页面

### 2.3 按钮样式配置

- **按钮名称：** 按钮的显示名称，最多可设置 **50 个字符**
- **按钮风格：** 可自定义按钮的展示风格
- **按钮颜色：** 可自定义按钮颜色

### 2.4 操作类型与显示位置

自定义按钮支持两大类操作类型：

#### 2.4.1 针对单条数据的操作

按钮展示在每条数据上，支持以下 **3 类显示位置**：

| 显示位置 | 说明 |
|---------|------|
| **数据详情页** | 默认勾选，不可取消。按钮显示在数据详情页右上角 |
| **视图数据列表** | 勾选后，按钮会在权限组的视图中显示（表格视图等） |
| **仪表盘数据管理表格** | 勾选后，按钮在仪表盘的数据管理表格中显示 |

#### 2.4.2 针对多条数据的批量操作

支持在表格视图中选择多条数据后，通过工具栏按钮批量执行操作。

### 2.5 可触发的动作类型（核心）

自定义按钮支持以下 **3 种主要执行动作**：

#### ① 新增数据

- **功能：** 点击按钮后，提供本表/他表的数据填报页面（根据按钮设置预先填充部分/全部字段内容），填写后直接提交
- **目标表单：** 可以是当前表单或其他表单
- **字段预填充：** 支持将当前数据的字段值映射到目标表单字段
- **关联子表数据同步：** 新增数据时，可将当前表单中关联子表的字段值同步新增到目标表单的关联子表字段中
  - 单次最多支持同步新增 **500 条**关联表数据
- **批量传递：** 支持在表格视图中选择多条数据后，通过按钮批量传递数据至关联子表
- **可用条件：** 支持设置按钮的可用条件（满足条件时按钮才可点击）

**典型场景：**
- 采购申请通过审批后 → 「发起订单」按钮 → 快速发起采购订单
- 确认客户意向后 → 「新建商机」按钮 → 快速新建商机
- 采购流程完成后 → 「入库」按钮 → 同步新增数据到入库表

**来源：** https://hc.jiandaoyun.com/doc/20400, https://hc.jiandaoyun.com/doc/22320

#### ② 编辑数据

- **功能：** 点击按钮时，提供**指定字段**的编辑页面，编辑内容并提交后更新该数据内对应字段的值
- **效果：** 与在本表中手动编辑数据一致，可触发数据联动、公式、表单校验等
- **权限要求：** 成员需具有相应数据的**编辑权限**
- **可用条件：** 支持设置按钮的可用条件

**典型场景：**
- 线索回退：「退回」按钮 → 直接将线索状态更新为"无效"
- 项目进度汇报：「进度汇报」按钮 → 快速更新项目进度信息
- 简历筛选：「淘汰」按钮 → 一键更新简历状态为"淘汰"

**来源：** https://hc.jiandaoyun.com/doc/20525

#### ③ 执行智能助手（Pro）

- **功能：** 点击按钮后触发智能助手Pro，自动执行预设的自动化流程
- **操作类型限制：** 仅支持「针对单条数据的操作」
- **智能助手Pro能力：** 
  - 新增/修改数据
  - 打开数据新增页
  - 打开数据多选页
  - 发送邮件通知
  - Agent节点
  - AI执行节点
  - 循环、调用API等

**典型场景：**
- 考勤点名：「点名签到」按钮 → 触发智能助手Pro → 自动将点名结果同步新增至签到记录表
- 任务领取：「领取任务」按钮 → 任务状态自动变更为"进行中" + 新增任务流程数据
- 发送发票：「发送发票」按钮 → 触发智能助手Pro → 自动将发票通过邮件发送给客户

**来源：** https://hc.jiandaoyun.com/doc/20396, https://hc.jiandaoyun.com/doc/21735

### 2.6 可用条件配置

所有类型的自定义按钮均支持设置**可用条件**：
- 当满足设定条件时，按钮才可被点击
- 不满足条件时，按钮置灰或隐藏
- 支持基于当前数据字段值的条件判断

**来源：** https://hc.jiandaoyun.com/doc/20400, https://hc.jiandaoyun.com/doc/20525, https://hc.jiandaoyun.com/doc/20396

## 三、弹窗功能

### 3.1 内置弹窗机制

简道云在以下场景中提供内置弹窗支持：

1. **数据编辑弹窗：** 在自定义详情页中点击「编辑」按钮，弹出数据编辑窗口
2. **数据新增弹窗：** 自定义按钮触发"新增数据"时，弹出数据填报页面
3. **数据多选弹窗：** 智能助手Pro中的「打开数据多选页」节点，弹出数据选择弹窗
   - 支持设置弹窗标题
   - 支持设置可选数据范围（过滤条件）
   - 支持将选择的数据用于后续节点

**来源：** https://hc.jiandaoyun.com/doc/22707, https://hc.jiandaoyun.com/doc/26005

### 3.2 页面弹窗插件

简道云插件市场提供「**页面弹窗**」插件（https://hc.jiandaoyun.com/open/16265），实现更灵活的弹窗功能：

- **功能：** 在表单填写时，以弹窗形式打开指定的链接内容
- **支持模式：** 
  - 单标签页弹窗
  - 多标签页弹窗（同时打开多个链接）
- **触发方式：** 通过按钮字段的执行动作触发
- **配置方式：** 在插件配置中设置链接地址

### 3.3 自建插件实现弹窗

通过自建插件的前端扩展函数，可以实现自定义弹窗：

- 使用 `$g.utils.openUrl` 对象打开标签页/弹窗
- 支持同时打开多个标签页
- 前端扩展函数可调用前端行为：弹出地图页面、录音、弹出 iframe 弹窗等

**代码示例（打开多个标签页）：**
```javascript
// 在前端扩展函数中使用 $g.utils.openUrl
$g.utils.openUrl(url1);
$g.utils.openUrl(url2);
```

**来源：** https://hc.jiandaoyun.com/open/23476, https://hc.jiandaoyun.com/open/16265, https://hc.jiandaoyun.com/open/23399

## 四、JavaScript API 与全局变量

### 4.1 前端事件（Frontend Events）

前端事件是简道云提供的**无需编程**的数据获取和交互方式：

- **设置入口：** 表单编辑页面 → 右侧工具栏 → 「表单属性 >> 前端事件」
- **免费使用：** 所有版本皆可免费使用，无需开通简道云API接口
- **触发时机：** 数据提交时或表单填报时
- **执行动作类型：**
  - **调用插件：** 调用市场插件或自建插件
  - **自定义请求：** 直接发起HTTP请求（GET/POST）

**来源：** https://hc.jiandaoyun.com/doc/11825, https://hc.jiandaoyun.com/open/12115

### 4.2 自定义请求（Custom Request）

在前端事件中配置自定义请求：

- **请求类型：** GET / POST
- **URL：** 请求的服务器地址。GET请求时参数放在URL中，支持插入字段给参数赋值
- **Header：** POST请求时支持设置Header参数（name + Value）
- **Body：** POST请求时支持设置Body参数
- **字段值引用：** 支持在URL/Header/Body中插入表单字段值

**字段类型与传值方式：**

| 字段类型 | 传值格式 |
|---------|---------|
| 图片 | 文本（图片URL），多文件URL空格分隔 |
| 附件 | 文本（附件URL），多文件URL空格分隔 |
| 手写签名 | 文本（附件URL） |

**来源：** https://hc.jiandaoyun.com/open/12206

### 4.3 前端扩展函数全局对象 `$g`

自建插件的前端扩展函数中可用的全局对象和API：

#### `$g.utils` 工具对象

| 方法 | 功能 |
|------|------|
| `$g.utils.openUrl(url)` | 打开指定URL（标签页/弹窗形式） |

#### 前端扩展函数能力

- 弹出地图页面
- 录音
- 弹出 iframe 弹窗
- 打开多个标签页
- 调用第三方API
- 数据回填到表单字段

### 4.4 自建插件体系

**插件设计逻辑：**
- 一个插件可包含一个或多个**函数**
- 每个函数包含 3 个属性：
  1. **代码**（后端函数 或 前端扩展）
  2. **请求参数声明**（入参声明）
  3. **返回参数声明**（出参声明）

**函数类型：**

| 类型 | 说明 |
|------|------|
| **后端函数** | 在服务端执行，可调用第三方API、进行数据处理等 |
| **前端扩展** | 在客户端执行，可调用前端行为（弹窗、打开页面等） |

**插件调用方式：**
- 通过前端事件调用
- 通过按钮字段的执行动作调用

**来源：** https://hc.jiandaoyun.com/open/16641, https://hc.jiandaoyun.com/open/23399, https://hc.jiandaoyun.com/open/23401

### 4.5 按钮字段（Button Field）

按钮字段是一种特殊的表单字段，与自定义按钮不同，它嵌入在表单中：

- **按钮文字：** 默认"按钮"，最多50个字
- **按钮样式：** 支持自定义按钮风格和颜色
- **执行动作：**
  - 调用插件（如企业信息查询插件）
  - 执行自定义请求（如手机号验真）
- **使用位置：** 主表和子表单中均可使用

**来源：** https://hc.jiandaoyun.com/doc/15581

## 五、代码示例汇总

### 5.1 前端扩展 - 打开多个标签页

```javascript
// 自建插件 - 前端扩展函数
// 功能：同时打开多个标签页

// 获取请求参数中的URL列表
const urls = params.urls; // 假设入参声明了urls数组

// 逐个打开标签页
urls.forEach(url => {
  $g.utils.openUrl(url);
});
```

### 5.2 自定义请求配置示例

```
请求类型: POST
URL: https://api.example.com/verify
Header:
  - name: Content-Type
    Value: application/json
  - name: Authorization
    Value: Bearer {{token}}
Body:
  {
    "phone": "{{手机号字段}}",
    "name": "{{姓名字段}}"
  }
```

### 5.3 自建插件 - 后端函数示例（计算最大值）

```javascript
// 入参：num_a, num_b, num_c
// 出参：max_value

const max = Math.max(params.num_a, params.num_b, params.num_c);
return { max_value: max };
```

### 5.4 页面弹窗插件配置

```
1. 安装「页面弹窗」插件（插件市场）
2. 启用插件（开放平台 >> 插件管理）
3. 在按钮字段的「执行动作」中选择「页面弹窗」
4. 配置弹窗URL地址
5. 点击按钮即可以弹窗形式打开对应链接
```

## 六、版本与权限说明

### 6.1 版本要求

- **自定义详情页：** 需升级到自定义详情页（从系统默认详情页升级）
- **前端事件：** 所有版本免费使用
- **自建插件：** 需要代码能力，建议无代码用户使用市场插件
- **AI相关功能：** AI实验室公测中，限时免费

### 6.2 权限控制

- 自定义按钮的使用受**数据权限**控制
- 编辑数据按钮要求成员具有**编辑权限**
- 自定义详情页打印时遵循**字段可见权限**
- 按钮可设置**可用条件**进一步控制

## 八、注意事项与限制

1. **SPA架构限制：** 简道云帮助文档网站为SPA架构，部分内容通过JavaScript动态加载，直接抓取可能无法获取完整内容
2. **关联子表同步上限：** 自定义按钮新增数据时，单次最多同步新增500条关联表数据
3. **阶段字段限制：** 设置了数据联动的下拉框/单选按钮组，或设置了关联其他表单的下拉框不支持作为阶段字段
4. **移动端同步：** 从电脑端创建移动端详情页时，除多标签页外的所有组件初始均为同步组件
5. **按钮名称长度：** 最多50个字符
6. **前端事件免费：** 所有版本均可免费使用前端事件功能
7. **自建插件需代码能力：** 若无代码能力，建议使用市场插件或提交需求
