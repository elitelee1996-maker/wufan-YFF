## 2. 简道云能力匹配矩阵 (Capability Matrix)

### 2.1 IoT 监控与实时看板

| 功能模块 | 标准功能覆盖 | 二开可实现性 | API/技术限制 | 实现思路摘要 |
| :--- | :--- | :--- | :--- | :--- |
| **IoT 协议支持** | ❌ 不支持原生 MQTT | ✅ 可行 (需中转) | 简道云不直接监听 MQTT Broker | 需部署外部 IoT 网关（如 EMQX）将 MQTT 转为 HTTP POST 调用简道云 API。 |
| **数据写入频率** | ⚠️ 受限 | - | API 限流：约 100-200 次/分 (视版本而定) | 高频传感器数据建议在外部数据库（如 InfluxDB）聚合后，按分钟级同步至简道云。 |
| **实时看板** | ✅ 支持 | - | 组件刷新频率最低通常为 1-5 分钟 | 通过设置仪表盘组件的“自动刷新”频率实现准实时监控；超实时需求需外接大屏系统。 |

*来源: [简道云 IoT 联动应用](https://www.jiandaoyun.com/news/article/6842becc6544d8159cd6f848), [API 文档](https://hc.jiandaoyun.com/open/11470)*

### 2.2 离线巡检与移动端

| 功能模块 | 标准功能覆盖 | 二开可实现性 | API/技术限制 | 实现思路摘要 |
| :--- | :--- | :--- | :--- | :--- |
| **离线录入** | ✅ 支持 (App 端) | - | 依赖简道云官方 App 或企业微信/钉钉集成 | 简道云移动端 App 已内置离线缓存机制，支持断网录入，联网后自动补传。 |
| **PWA/原生壳** | ⚠️ 部分支持 | ✅ 可行 (高级二开) | 浏览器 Service Worker 兼容性 | 若需独立 PWA，需自行开发 Shell 应用并处理本地 IndexedDB 缓存与 API 同步逻辑。 |
| **数据同步** | ✅ 自动同步 | - | 冲突处理由平台底层接管 | 建议在设计表单时增加“离线标记”字段，便于后续追踪数据来源。 |

*来源: [离线功能设计指南](https://www.jiandaoyun.com/news/article/6a01926b34594e4ba69760b6), [MES 移动端评估](https://www.jiandaoyun.com/news/article/68a85791229b892d52d07078)*

### 2.3 ERP 对接与主数据同步

| 功能模块 | 标准功能覆盖 | 二开可实现性 | API/技术限制 | 实现思路摘要 |
| :--- | :--- | :--- | :--- | :--- |
| **API 网关** | ✅ 提供 RESTful API | - | 需企业版及以上；密钥对管理 | 使用简道云开放平台 API 进行表单数据的增删改查；支持一对密钥管理。 |
| **Webhook 触发** | ✅ 支持 | - | 仅支持特定事件（新增、修改等） | 配置 Webhook 回调地址，实现 ERP 下单后自动触发简道云库存扣减或工单生成。 |
| **主数据同步** | ⚠️ 需定时任务 | ✅ 可行 | 避免循环触发；注意 API 频次 | 建议采用“中间表”策略或在外部部署调度器（如 Python 脚本）定期拉取/推送主数据。 |

*来源: [ERP 接 API 详解](https://www.jiandaoyun.com/nblog/105824), [数据推送 Webhook](https://hc.jiandaoyun.com/open/11500)*

## 3. 综合风险提示

- **技术风险**: 
    - **IoT 高频数据瓶颈**: 简道云 API 存在调用频次限制，直接接入高频传感器数据会导致请求失败。**建议**: 必须引入边缘计算网关或时序数据库进行数据清洗和聚合。
    - **离线冲突**: 虽然 App 支持离线录入，但在多端同时修改同一条设备档案时，可能存在最后写入覆盖的风险。**建议**: 业务设计上尽量避免多人同时编辑同一设备记录。

- **实施风险**: 
    - **ERP 对接复杂度**: 主数据（如物料、设备编码）在 ERP 与简道云之间的对齐是实施难点。**建议**: 在项目初期确立“单一数据源”原则，明确哪一方为主数据维护端。
    - **网络环境依赖**: 尽管有离线功能，但复杂的关联查询和报表分析仍强依赖网络。**建议**: 在工厂弱网区域提前进行 App 缓存压力测试。
