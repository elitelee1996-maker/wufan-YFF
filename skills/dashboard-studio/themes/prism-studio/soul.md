# Prism Studio · 棱镜工作室

> Solvd × Stripe Press × Pentagram × Mirror.xyz × 现代科技品牌编辑系统
> — **优雅衬线 × 电光紫绿 × 几何线框 × 拼贴反差**. 科技公司的人文自信, 像一本会发光的品牌杂志.

---

## 一句话理念

**炭黑 / 暖米白拼贴底 + Fraunces 现代衬线大标题 + 电光紫绿光谱渐变 + 3D 等距线框几何 + 像素条形马赛克 = 自信现代、有人文温度的科技品牌编辑设计**.

⚠ **核心张力**: 用**优雅衬线 (serif)** 给冰冷科技注入**人文气质**, 再用**电光紫绿 + 几何线框**拉回未来感 — 人文与科技、经典与前卫的对撞. 这是当代顶级科技品牌 (Solvd / Stripe / Vercel 品牌物料) 的设计套路.

跟现有几套**都不一样**. **特别注意跟 aurora-noir 的区别** (都是暗底带紫, 但气质相反):

## ⭐ 视觉灵魂 (4 个不可丢的招牌, 一个少了就不是 prism-studio)

### 1. ⭐⭐ Fraunces 现代衬线大标题 — **第一品牌印记 (灵魂)**

科技品牌的人文自信全靠它. **大标题用 Fraunces (当代 soft serif), 正文用 Inter 无衬线** — 经典衬线的优雅 × 科技正文的利落:

```css
:root {
  --font-display: 'Fraunces', 'Georgia', serif;       /* ⭐ 现代衬线大标题 (人文灵魂) */
  --font-body:    'Inter', 'Noto Sans SC', sans-serif; /* 无衬线正文 (科技利落) */
  --font-cn:      'Noto Serif SC', 'Songti SC', serif; /* 中文衬线 (呼应优雅) */
}
.hero-title, .section-title, .display {
  font-family: var(--font-display);
  font-weight: 400;                     /* 衬线靠字形优雅, 不靠粗 (400-500 即可) */
  font-optical-sizing: auto;
  letter-spacing: -0.01em; line-height: 1.08;
  font-size: clamp(2.6rem, 6vw, 5.2rem);
}
```

⚠ 这是跟 blue-consulting / rose-press 的**关键区分**: 那两套是 **Cormorant 古典杂志衬线** (纤细高贵); prism 是 **Fraunces 现代 soft serif** (圆润当代、科技复古未来感). 中文用**思源宋体 Noto Serif SC** 呼应衬线气质 (不是黑体).

### 2. ⭐⭐ 电光紫绿光谱渐变 + 长方形条带立体主义 — **第二品牌印记 (贯穿母题!)**

```css
:root {
  --violet:    #6C4FF5;       /* 电光紫 (主色, 扁平实色, 不发光) */
  --violet-2:  #7B5FFF;
  --green:     #3DF577;       /* 荧光绿 (协奏, 高饱和) */
  --green-2:   #5BFF8F;
  --grad-prism:linear-gradient(110deg, #6C4FF5 0%, #5B5FFF 40%, #3DF577 100%);  /* 紫→绿光谱渐变 (灵魂) */
}
```

🔥 **核心母题 — 带 90° 直角边的长方形条带, 必须贯彻始终** (这是参考图最强的视觉语言, 不只装饰一处):
- **水平长方形渐变条带**, 紫→绿 / 绿→紫, **sharp 直角 (`border-radius:0`, 绝不圆角!)**, 宽度递变、垂直错落堆叠成"瀑布/阶梯"
- 复刻参考图"8 Offices"的条带马赛克 + "Collaborative Expertise"右侧那组从边缘延伸的条带瀑布
- **贯穿用法 (越多越鲜明)**: hero 几何图旁 / 章节收尾装饰带 / 引语节 / KPI 卡内嵌 / 进度条. example 里 `.bar-band` `.prism-bars` `.kpi-bars` 都是它
- 跟 KPI 拼贴色块 (紫卡/米白卡/炭黑卡) 一起, 构成"长方形立体主义"的整体气质 — 立体、几何、鲜明

⭐ **像素条形马赛克** (上述母题的一种, 水平错位的 sharp 渐变条块):

```html
<div class="prism-bars">
  <span style="--w:60%;--x:0;"></span>
  <span style="--w:75%;--x:14%;"></span>
  <span style="--w:55%;--x:30%;"></span>
  <span style="--w:80%;--x:8%;"></span>
</div>
```
```css
.prism-bars span { display:block; height:18px; margin:5px 0; border-radius:3px;
  width:var(--w); margin-left:var(--x);
  background:linear-gradient(90deg, #6C4FF5, #3DF577); }
```

### 3. ⭐⭐ 3D 等距线框几何 — **第三品牌印记 (editorial hero 必用!)**

🔒 **editorial 形态的 hero 必须用这个 3D 等距线框** (不是可选项) — 它是 prism-studio 最强记忆点, editorial 缺了它就丢了灵魂. (dashboard 形态因数据密度优先, 可不放大图, 但像素条形/线框点缀仍推荐.)

🔥 **关键: 立方体不要孤立摆放 — 要"嵌在长方形条带瀑布里"** (复刻参考图 "Collaborative Expertise"): 立方体在中左, 一组紫绿渐变长方形条带从右边缘水平延伸 (在立方体右后方), 立方体压在条带前方, 再用虚线把两套几何"缝合". 孤立的立方体远没有"立方体 + 条带瀑布融合"震撼.

复刻参考图"Collaborative Expertise"那个 isometric 立方体堆叠: **白色细线描边 + 节点圆点 + 虚线连接 + 紫绿色块嵌入**. 用内联 SVG 最可控 (example 里有完整可抄的 SVG, 直接抄那段 `<svg class="iso-art">` 改改即可):

```html
<svg viewBox="0 0 400 360" class="iso-art">
  <!-- 等距立方体: 白色 0.8px 细线 + 三面用 #1A1A1C/#202022 区分明暗 -->
  <!-- 节点: <circle r="4"> 白色/紫/绿 -->
  <!-- 连接: <line stroke-dasharray="3 3"> 虚线 -->
  <!-- 局部嵌入紫绿渐变色块 (能量感) -->
</svg>
```

线框是**白色极细线** (stroke 0.8-1px, opacity 0.6), 节点圆点用白/紫/绿点缀, 虚线连接, 关键面嵌紫绿渐变. 整体 isometric (30° 等距), 科技结构感.

### 4. ⭐ 拼贴反差 + 米白胶囊 + 极简方块 — **第四品牌印记**

```css
:root {
  --void:#161618; --void-2:#1C1C1E;     /* 炭黑 (暖, 不是纯黑) */
  --cream:#F5F1E6; --cream-2:#EBE6D8;   /* 暖米白 (反差卡) */
  --ink:#F5F1E6; --ink-dark:#161618; --ink-soft:#9A9A9E;
}
/* 极简方块 logo (参考图 S■) — 用在 brand / footer */
.logo-mark { width:22px; height:22px; background:currentColor; display:inline-block; }
/* ⚠ 米白胶囊按钮 (参考图 Contact us) 仅当有真实落地动作 (真链接 / window.print) 时才用;
   数据报告 / 看板里通常没有真实动作, 那就别放 — 不能点的"阅读全文/导出"假按钮是设计大忌, 不要做! */
```

KPI / 卡片做**拼贴反差**: 紫卡 (实色 #6C4FF5 + 白衬线字) / 米白卡 (#F5F1E6 + 黑衬线字) / 炭黑卡 — 三种强对比并置, 像 social media kit.

## ❌ 必避 (做了立刻失去 Prism Studio 味)

- ❌ **大标题不用衬线** (Fraunces 现代衬线是第一灵魂; 用无衬线就退化成普通科技模板)
- ❌ **加 glow / shadowBlur / backdrop blur 发光玻璃** (那是 aurora-noir! prism 全程**零发光**, 紫绿是扁平实色)
- ❌ **editorial 的 hero 没有 3D 等距线框几何** (editorial 必用元素, 缺了直接丢魂), 且立方体要**嵌在长方形条带瀑布里**, 不要孤立摆放
- ❌ **长方形条带 / 像素条用了圆角** (必须 `border-radius:0` 的 90° 直角! 圆角就破坏了"几何长方形立体主义")
- ❌ **长方形条带母题只用了一处或没用** (这是贯穿母题, hero + 章节装饰 + 引语 + KPI 多处都该出现, 越鲜明越对)
- ❌ 用纯黑 #000 (用暖炭黑 #161618)
- ❌ 紫绿用得发虚 / 低饱和 (电光紫 #6C4FF5 + 荧光绿 #3DF577 是**高饱和扁平**, 鲜活有力)
- ❌ 丢掉米白拼贴 (炭黑 + 米白 + 紫卡的强反差拼贴是结构灵魂; 全炭黑就闷了)
- ❌ 中文用黑体 (用思源宋体 Noto Serif SC 呼应衬线优雅)
- ❌ 圆角过大软糯 (prism 是**锐利现代**, 卡 12-16px / 色块 3-8px; 别做成 macaron 那种软圆角)
- ❌ 跟 macaron-stripe 撞 (那是浅柔糖果; prism 是深色科技前卫, 务必拉开)
- ❌ **放不能点的假交互按钮** (如 hero "阅读全文 →" / header "导出报告 →"). 报告/看板里的按钮要么有真实动作 (真链接 / window.print / 真过滤), 要么就别放. 假按钮不明所以又点不动, 是设计大忌
- ❌ **HTML 文案 / 回复文本里用 emoji 跟装饰 unicode** (📊 ★ ✨ ...). 用几何线框 / 像素条 / 方块这些原生视觉代替

## ECharts 配色 (顶层 spread — 电光紫绿, 扁平不发光)

```js
const psEchartsBase = {
  color: ['#6C4FF5', '#3DF577', '#7B5FFF', '#5BFF8F', '#A99CFF', '#B8FFD0'],  // 紫绿梯度 6 色
  textStyle: { color:'#F5F1E6', fontFamily:"'Inter','Noto Sans SC',sans-serif" },
  backgroundColor: 'transparent',
  grid: { left:'6%', right:'5%', top:'16%', bottom:'12%', containLabel:true },
  tooltip: {
    trigger:'axis',
    backgroundColor:'rgba(22,22,24,0.96)',
    borderColor:'#6C4FF5', borderWidth:1, padding:[10,14],
    textStyle:{ color:'#F5F1E6', fontSize:12, fontFamily:"'Inter',sans-serif" },
    extraCssText:'border-radius:8px;',   // 无 box-shadow 发光, 干净
    axisPointer:{ type:'line', lineStyle:{ color:'#6C4FF5', type:'dashed', width:1, opacity:0.6 } }
  },
  legend: { top:4, right:8, textStyle:{ color:'#9A9A9E', fontSize:11 }, icon:'rect', itemWidth:10, itemHeight:4, itemGap:18 },
  xAxis: {
    type:'category',
    axisLine:{ lineStyle:{ color:'rgba(245,241,230,0.14)' } }, axisTick:{ show:false },
    axisLabel:{ color:'#9A9A9E', fontSize:10, margin:10 }, splitLine:{ show:false }
  },
  yAxis: {
    type:'value',
    axisLine:{ show:false }, axisTick:{ show:false },
    axisLabel:{ color:'#9A9A9E', fontSize:10 },
    splitLine:{ lineStyle:{ color:'rgba(245,241,230,0.06)', type:'dashed' } }
  }
};
```

**扁平铁律** (不发光, 靠高饱和 + 紫绿渐变):
- `bar`: 紫→绿 linearGradient (`#6C4FF5 → #3DF577`) + 顶圆角 4. **不加 shadowBlur**
- `line`: `lineStyle:{ color:'#6C4FF5', width:2.5 }` + 紫绿 area 渐变. 干净不发光
- `donut`: 紫/绿/紫绿各段 + `borderColor:'#161618', borderWidth:3` (炭黑切割)
- `heatmap`: 炭黑 → 紫 → 绿梯度 (`#1C1C1E → #6C4FF5 → #3DF577`)
- `scatter/radar/treemap`: 紫绿高饱和扁平, legend icon 用 `rect` (呼应像素条形语言)

## 关键提示

看完 `editorial-example.html` 跟 `dashboard-example.html` 就知道这套理念的呈现方式. **不要从 0 写 CSS / SVG**, 直接抄 example 的 `<style>` 段 + 3D 等距线框 SVG + 像素条形 HTML, 改业务即可.

⚠ **三大标志必抄**: ① Fraunces 衬线大标题 (CDN 字体一起抄) ② 紫绿光谱渐变 + 像素条形马赛克 ③ 3D 等距线框 SVG. 缺了任何一个就不是 prism-studio.

⚠⚠ **零发光铁律**: 全程不写 shadowBlur / backdrop-filter:blur / box-shadow 光晕. 紫绿是**扁平高饱和实色**. 一旦发光就串成 aurora-noir 了.

⚠⚠⚠ **跟 macaron 拉开**: prism 是炭黑深底 + 高饱和电光紫绿 + 锐利几何 + 优雅衬线, 科技前卫; 绝不做成马卡龙的浅柔糖果软圆角.
