# Kinetic Noir · 动感暗夜

> Nike Run Club × Whoop × Strava × Linear (Plasma) × Apple Fitness
> — **运动品牌的能量暗夜**. 不是奢华金融,不是赛博电影 — 是黑底上烧着烈橙火焰的运动 banner.

---

## 一句话理念

**纯黑深底 + 烈橙红 #FF6B2E 主色 + Anton 极粗工业大写字 + 超大软圆角卡片 + 悬浮数据 pill + 橙→透明 area 渐变 = 直接有力、有温度的运动品牌质感**.

⚠ 核心调性: **简洁、直接、有运动能量** — 不发光、不奢华、不复杂. 烈橙是温度,极粗大写字是力量,大软圆角是亲和,悬浮 pill 是动感. 像 Nike Run Club / Apple Fitness 那种你看一眼就想动起来的暗夜.

---

- kinetic = 黑底 + **烈橙** + **极粗大写字** + **大软圆角** → 像 **Nike 训练 app**

- aurora = 蓝黑 + **紫青霓虹** + **玻璃发光** → 像 **Web3 dashboard**

三套互不串色, 三种暗夜业务场景泾渭分明.

## ⭐ 视觉灵魂 (4 个不可丢的招牌, 一个少了就不是 kinetic-noir)

### 1. ⭐⭐ Anton 极粗 condensed 工业大写字 — **第一品牌印记 (最强识别!)**

参考图 "TOP TASKS" 那种**窄高、极粗、紧凑、全大写**的工业 banner 体 — 这是 Druk Wide / Anton / Bebas Neue 这一类的灵魂. 主标题必用 Anton (Google Fonts 免费替代 Druk), 数字用 Inter Tight 800/900 极粗:

```css
:root {
  --font-display: 'Anton', 'Bebas Neue', sans-serif;        /* ⭐ 极粗 condensed 大写 — 灵魂 */
  --font-numeric: 'Inter Tight', 'Inter', sans-serif;       /* 大数字 800/900 */
  --font-body:    'Inter', 'Noto Sans SC', sans-serif;
  --font-cn:      'Noto Sans SC', 'PingFang SC', sans-serif; /* 中文用黑体, 配运动感 */
}
.hero-title, .section-title, .display-title {
  font-family: var(--font-display);
  font-weight: 400;            /* Anton 本身就是极粗, weight 400 已经够冲击 */
  text-transform: uppercase;    /* 必大写! */
  letter-spacing: -0.005em;     /* 紧凑 */
  line-height: 0.95;            /* 行距紧凑, 像 banner */
  font-size: clamp(3rem, 7vw, 6rem);
}
.big-number, .kpi-value {
  font-family: var(--font-numeric);
  font-weight: 800;
  letter-spacing: -0.03em;
  font-variant-numeric: tabular-nums;
}
```

⚠ 这是跟 ember-noir (Sora 几何) / aurora-noir (Montserrat) / blue-consulting (Cormorant 衬线) 的**核心区分** — 只有 kinetic 用 condensed 工业大写字. 中文标题保持思源黑体, **不要用衬线** (会破坏运动感).

### 2. ⭐⭐ 烈橙红 #FF6B2E + 纯黑中性 — **第二品牌印记**

```css
:root {
  --void:        #0E0E0E;       /* 纯黑深底 (中性, 不暖不冷) */
  --surface:     #1A1A1A;       /* 卡片 */
  --surface-2:   #252525;       /* 抬升 / hover */
  --ember:       #FF6B2E;       /* ⭐ 烈橙红 (主色, 温度直接, 不是金) */
  --ember-soft:  #FF8A5C;       /* 浅橙 (area 渐变 / 副) */
  --ember-deep:  #C84B0C;       /* 深橙 (悬浮 hover) */
  --ink:         #F5F5F5;       /* 主白文字 */
  --ink-soft:    #A0A0A0;       /* 次灰 */
  --ink-faint:   #666666;       /* 弱 */
  --hair:        rgba(255,255,255,0.08);  /* 中性灰白细线, 不带橙染 */
}
body { background: var(--void); color: var(--ink); }
```

**铁律**: **没有金、没有香槟、没有紫青、没有衬线、没有发光**. 大面积是黑 + 灰, 烈橙做强调色 (按钮 / 高亮线 / area / pill / 激活态), 其他全部中性. 比 ember (面积 < 10%) 的金可以**用得更大胆**一点(area 渐变可以铺得很大), 因为烈橙本身就是品牌主调而非点睛.

### 3. ⭐ 超大软圆角卡片 + 圆形元素 — **第三品牌印记**

```css
:root { --r-card: 28px; --r-pill: 999px; --r-btn: 50%; }
.card, .kpi, .panel { border-radius: var(--r-card); background: var(--surface); }
.icon-btn-round { width: 48px; height: 48px; border-radius: 50%; background: var(--surface); display: inline-flex; align-items: center; justify-content: center; }
.avatar { width: 48px; height: 48px; border-radius: 50%; overflow: hidden; }
.dot-legend { width: 8px; height: 8px; border-radius: 50%; }
.pill-pop { background: #fff; color: #0E0E0E; padding: 8px 16px; border-radius: 14px; font-weight: 700; font-family: var(--font-numeric); box-shadow: 0 8px 20px rgba(0,0,0,0.5); }
```

软糯、现代、运动 app 感. 跟 ember (14px 小圆角) / aurora (8px 极小) 完全不同 — kinetic 是**大软** + **圆**.

### 4. ⭐⭐ 悬浮数据 pill + 橙→透明 area 渐变 chart — **第四品牌印记 (运动数据感的灵魂)**

复刻参考图最有记忆点的元素: **折线图上漂浮一颗白底 pill "30% ↑"**, 配橙色 area 渐变. 这是运动 app 数据可视化的标志手法:

```js
// 折线 + 渐变 area (橙色品牌色 + 灰白对照)
series: [{
  name: '主线',
  type: 'line', smooth: true, symbol: 'circle', symbolSize: 0,
  data,
  lineStyle: { color: '#FF6B2E', width: 2.5 },     // 不加 shadowBlur (绝不发光!)
  areaStyle: { color: { type: 'linear', x: 0, y: 0, x2: 0, y2: 1, colorStops: [
    { offset: 0, color: 'rgba(255,107,46,0.55)' },
    { offset: 1, color: 'rgba(255,107,46,0)' },
  ]}}
}, {
  name: '对照',
  type: 'line', smooth: true, symbol: 'none', data: data2,
  lineStyle: { color: 'rgba(245,245,245,0.85)', width: 1.5, type: [4,4] },  // 灰白虚线
}]
// 加 markPoint 实现悬浮 pill "30% ↑" (灵魂)
markPoint: { data: [{ coord: [peakX, peakY], value: '30% ↑' }],
  symbol: 'roundRect', symbolSize: [60, 28], symbolOffset: [0, -22],
  itemStyle: { color: '#fff' },
  label: { color: '#0E0E0E', fontWeight: 700, fontFamily: "'Inter Tight'", fontSize: 13 }
}
```

ECharts 主图必须出现这个组合(area + 悬浮 pill 标注), 是 kinetic 的视觉签名.

## ❌ 必避 (做了立刻失去 Kinetic Noir 味)

- ❌ **大标题不用 Anton/Bebas/Anton 那类极粗 condensed 大写字** (这是第一灵魂; 用 Sora/Fraunces/Cormorant 立刻退化成别的主题)
- ❌ **用金色 / 香槟色 / 衬线字 / 玑镂磨砂** (那是 ember-noir 的; kinetic 是直接的烈橙 + 纯净中性)
- ❌ **加 glow / shadowBlur / backdrop blur / 极光** (那是 aurora-noir 的; kinetic 全程零发光)
- ❌ **没有橙 area 渐变 chart 或 没有悬浮数据 pill** (两大签名图形至少出现一次, 推荐都有)
- ❌ 用纯黑 #000 (用 `#0E0E0E` 中性纯黑) / 暖黑 (那是 ember)
- ❌ 用小圆角 / sharp 棱角 (kinetic 是大软圆角 24-32px + 圆形元素)
- ❌ 橙色用得发虚 / 暗淡 (烈橙 `#FF6B2E` 是**饱和直接**的, 不要降低饱和)
- ❌ 中文标题用衬线 (用黑体, 跟工业大写字呼应运动感)
- ❌ 主标题混排大小写 (主标题必全大写, banner 感; 正文正常排版)
- ❌ **HTML 文案 / 回复文本里用 emoji 跟装饰 unicode** (📊 ★ ✨ ...). 用大写字 / 烈橙圆点 / 圆形按钮等原生视觉代替

## ECharts 配色 (顶层 spread — 烈橙 + 灰白对照, 扁平不发光)

```js
const knEchartsBase = {
  color: ['#FF6B2E','#F5F5F5','#FF8A5C','#999999','#C84B0C','#666666'],  // 烈橙梯度 + 灰白对照
  textStyle: { color:'#F5F5F5', fontFamily:"'Inter','Noto Sans SC',sans-serif" },
  backgroundColor: 'transparent',
  grid: { left:'6%', right:'5%', top:'16%', bottom:'12%', containLabel:true },
  tooltip: {
    trigger:'axis',
    backgroundColor:'rgba(14,14,14,0.96)',
    borderColor:'#FF6B2E', borderWidth:1, padding:[10,14],
    textStyle:{ color:'#F5F5F5', fontSize:12, fontFamily:"'Inter Tight','Inter',sans-serif" },
    extraCssText:'border-radius:14px;',           // 无 box-shadow 发光, 干净
    axisPointer:{ type:'line', lineStyle:{ color:'#FF6B2E', type:'dashed', width:1, opacity:0.5 } }
  },
  legend: { top:4, right:8, textStyle:{ color:'#A0A0A0', fontSize:12, fontWeight:500 }, icon:'circle', itemWidth:8, itemHeight:8, itemGap:18 },
  xAxis: {
    type:'category',
    axisLine:{ lineStyle:{ color:'rgba(255,255,255,0.1)' } }, axisTick:{ show:false },
    axisLabel:{ color:'#A0A0A0', fontSize:11, margin:10 }, splitLine:{ show:false }
  },
  yAxis: {
    type:'value',
    axisLine:{ show:false }, axisTick:{ show:false },
    axisLabel:{ color:'#A0A0A0', fontSize:11 },
    splitLine:{ lineStyle:{ color:'rgba(255,255,255,0.06)', type:'dashed' } }
  }
};
```

**扁平铁律** (跟 ember/aurora 严格区分: 不发光, 不渐变满屏, 干净直接):
- `line` (主签名): 烈橙线 2.5px + 橙→透明 area 渐变 + 灰白虚线对照 + **markPoint 悬浮 pill**. 绝不加 shadowBlur
- `bar`: 实色烈橙 + 顶圆角 6-8px, 高亮一根用浅橙 #FF8A5C 或加白边. 不加 shadowBlur
- `donut`: 烈橙 + 灰白 + 浅橙各段 + `borderColor:'#0E0E0E', borderWidth:3` (黑切割, 干净). 不发光
- `heatmap`: 黑 → 深橙 → 烈橙 → 浅橙梯度 (`#1A1A1A → #C84B0C → #FF6B2E → #FF8A5C`)
- `scatter/radar/treemap`: 烈橙为焦点高亮, 灰白为对照, 第三色用浅橙. 无金、无紫、无衬线 tooltip

## 关键提示

看完 `editorial-example.html` 跟 `dashboard-example.html` 就知道这套理念的呈现方式. **不要从 0 写 CSS**, 直接抄 example 的 `<style>` 段, 改业务即可.

⚠ **三大标志必抄**:
1. Anton 极粗 condensed 大写主标题 (字体 CDN 跟抄)
2. 橙→透明 area 渐变 + 悬浮 pill markPoint 的折线主图
3. 超大软圆角卡片 + 圆形头像/按钮

⚠⚠ **零发光铁律**: 全程不写 shadowBlur / backdrop-filter:blur / box-shadow 光晕. 烈橙是**直接饱和实色**. 一发光就退化成廉价炫光.

⚠⚠⚠ **绝不串到 ember/aurora**: kinetic 全程只用烈橙 + 中性灰白, 没有金、香槟、紫、青. 一旦出现这些, 就破坏了运动品牌的能量感.
