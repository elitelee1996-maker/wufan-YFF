# 视觉风格情绪板 · Moodboards

> **9 套顶尖品质的视觉哲学** — 不是选风格菜单, 是看清"什么叫顶尖前端设计"的对照组.

## 真正的价值

每套情绪板的目标**不是让你照抄**, 而是让你看清:

- **什么叫顶尖品质** — 字体粗细的微妙、间距的呼吸、色彩的克制、纹理的层次, 这些不可名状的"高级感"是怎么从细节累积出来的
- **调性的具象抓手** — 「暗夜霓虹」「杂志高级感」「运动能量」这些抽象词, 用代码片段说清楚
- **超越的起点** — 看完一套, 你应该有底气**做得跟它一样好, 或者比它更好, 或者完全不同但同样高品质**

## 风格不绑场景

任何一套都能套到任何场景上 — 只要调性匹配、执行到位:

- aurora-noir 的暗夜霓虹也可以做顶级邀请函 (科技公司产品发布)
- rose-press 的杂志感也可以做软件落地页 (创意工具品牌站)
- macaron-stripe 的马卡龙也可以做技术报告 (青少年编程教育)

风格跟场景的搭配是**判断题不是填空题**. 看你想表达什么气质, 不是看内容类别.

**也不必须用任何一套**. 完全可以原创、可以混搭、可以做大改 — 关键是品质对齐这里的水准.

## 怎么用

3 种用法, 自己挑:

1. **对齐品质** (主用法) — 读 1-2 篇看"细节是怎么讲究的", 学到工艺标准, 用到自己的原创里
2. **整套借鉴** — 调性契合就直接搬色彩 / 字体 / 关键质感
3. **混搭借鉴** — 多套各取所长 (graphic-press 的胶囊 × luminous-paper 的清爽 × aurora 的玻璃卡, 揉一个新的)

## 顶尖品质的共同底线

不论用不用风格, 这些是**所有 9 套都做到的基础品质** — 也是"顶尖前端"的水准线:

- **字体有性格** — 不用 Arial / 系统字体, 选 Cormorant / Fraunces / Anton / Montserrat / Space Grotesk / Sora 这种有辨识度的; 中文用 Noto Sans/Serif SC
- **色彩有观点** — 主色 1-2 个 + 辅助色 3-4 个, 不平摊 8 种颜色. 敢"少色"敢"撞色", 不敢"全收"
- **空间会呼吸** — 大量留白 / 强对比密度差. 别把内容塞满
- **细节经得起放大** — hairline 0.5px 不是 1px、圆角 12 不是 8、letter-spacing -0.02em 而不是 0 — 这些不可名状的细节决定"高级"
- **背景不是纯色** — 渐变 / dot-grid / 噪点 / 雾染 / 极光, 总之有"质感"
- **零 emoji** — 顶尖设计只用文字 / 图标 / 排版本身做强调. 这是铁律
- **动效有目的** — 要做就做精彩 (页面加载 stagger、关键悬停、滚动揭示), 不要零散点缀
- **单一焦点** — 每屏只有一件事最重要, 别同时强调 5 件事

**真正的本事**: 看完 9 套, 在做任何前端时, 都对"高品质"有具象标尺 — 知道距顶尖水平有多远, 怎么再推一推就到那里, 或者超越.
