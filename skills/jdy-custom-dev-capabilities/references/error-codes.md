# 简道云 API 错误码速查表

> 评估和调试时的快速参考

---

## 1. HTTP 状态码

| 状态码 | 含义 | 处置建议 |
|--------|------|---------|
| 2xx | 成功 | — |
| 400 | 接口错误 | 检查请求参数，返回体含 `{code, msg}` |
| 429 | 超出并发限制 | 降低请求频率，增加重试间隔 |
| 502 | 网关异常 | 稍后重试 |
| 579 | 文件上传失败 | 检查 token 是否过期，重新获取 |

---

## 2. 通用错误码

| 错误码 | 说明 | 处置建议 |
|--------|------|---------|
| 4001 | 参数错误 | 检查请求体格式和必填参数 |
| 4002 | API Key 无效 | 检查 Authorization Header |
| 4003 | 无权限访问该资源 | 检查 API Key 授权范围 |
| 4004 | 资源不存在 | 检查 app_id / entry_id / data_id |

---

## 3. 流程错误码

| 错误码 | 说明 | 评估影响 |
|--------|------|---------|
| 4005 | 操作失败，不允许撤回 | 检查流程节点是否支持撤回 |
| 4007 | 没有流程处理人 | 流程配置问题，需检查节点负责人设置 |
| 4008 | 流程已经关闭 | 无法对已结束流程操作 |
| 4009 | 无权限 | 操作人不是当前节点处理人 |
| 4016 | 流程不能转交给自己 | 转交目标人不能是当前操作人 |
| 5003 | 流程节点不存在 | 检查 flow_id 是否正确 |
| 5004 | 未提交流程审批意见 | 某些节点要求必须填写审批意见 |
| 5008 | 找不到对应的负责人 | 节点负责人配置问题 |
| 5010 | 无效的流程数据 | 检查 data_id 是否为流程表单数据 |
| 5012 | 未提交流程手写签名 | 节点要求手写签名 |
| 5024 | 当前用户对该节点无提交权限 | 检查节点权限配置 |
| 5049 | 当前流程节点未开启转交 | 节点配置不支持转交 |
| 5056 | 不能对非流程表单进行流程操作 | 确认表单类型 |
| 50016 | 流程实例不存在 | 检查 instance_id |
| 50034 | 不支持的流程操作类型 | 检查操作是否在节点允许范围内 |
| 50052 | 节点不支持该加签行为 | 检查节点加签配置 |
| 50054 | 不支持嵌套加签 | 已加签的节点不能再加签 |

### 评估时的流程错误码检查要点

| 需求场景 | 需关注的错误码 | 评估建议 |
|---------|---------------|---------|
| API 自动审批 | 4009, 5024 | 确认 API 操作人是否有节点权限 |
| 批量回退流程 | 4008, 5003 | 确认流程状态和目标节点 |
| 自动转交 | 4016, 5049 | 确认转交规则和节点配置 |
| 加签场景 | 50052, 50054 | 确认节点是否支持加签，避免嵌套 |

## 4. 通讯录错误码

| 错误码 | 说明 | 处置建议 |
|--------|------|---------|
| 6001 | 成员不存在 | 检查 username |
| 6002 | 成员已存在 | 避免重复创建 |
| 6003 | 部门不存在 | 检查 dept_no |
| 6004 | 部门已存在 | 避免重复创建 |
| 6005 | 不能删除根部门 | 根部门 dept_no=1 不可删除 |
| 6006 | 部门下有成员或子部门 | 需先移除成员和子部门 |

## 5. 数据操作错误码

| 错误码 | 说明 | 处置建议 |
|--------|------|---------|
| 7001 | 数据不存在 | 检查 data_id |
| 7002 | 字段不存在 | 检查 widgetName |
| 7003 | 字段类型不匹配 | 参照 field-type-mapping.md |
| 7004 | 批量操作超过 100 条 | 分批处理 |
| 7005 | transaction_id 不匹配 | 确保上传和数据操作使用同一 transaction_id |

## 6. 频率限制相关

| 场景 | 限制 | 计算示例 |
|------|------|---------|
| 全局限流 | 50 次/秒 | 所有接口共享 |
| 数据查询 | 30 次/秒 | 查 10000 条（每次 100 条）≈ 3.3 秒 |
| 数据写入（单条） | 20 次/秒 | 写 1000 条 ≈ 50 秒 |
| 数据写入（批量） | 20 次/秒 | 写 10000 条（每次 100 条）≈ 10 秒 |
| 文件上传 | 20 次/秒 | 上传 500 个文件 ≈ 25 秒 |
| 表单结构查询 | 30 次/秒（V5）/ 5 次/秒（V1/V2） | 推荐用 V5 |
| 通讯录操作 | 30 次/秒 | — |

### 大批量操作耗时估算公式

```
总耗时 ≈ (总数据量 / 每批条数) / 接口频率 + 处理开销

示例：同步 50000 条数据到简道云
= (50000 / 100) / 20 + 网络开销
= 500 / 20 + ~5s
= 25 + 5
≈ 30 秒
```

> ⚠️ 注意全局限流 50 次/秒。如果同时有其他 API 调用，实际可用频率更低。
